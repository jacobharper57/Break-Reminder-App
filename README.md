# Break-Reminder-App
A simple Swift UI application - for a college project.  Rose State - Made by Jacob Harper
Break Reminder App

#Overview
This app is a simple SwiftUI application that helps users maintain a reminder for taking breaks while working.
Users can configure how often they would like to take a break, and for how long.
Once the app is started and a schedule is set, when the timer reaches zero it will send a notification to remind the user to take a break for the set amount of time.
NOTE: Notifications will need to be enabled for the application to send push notifications.

#Features
-Set break frequency
-Set break duration
-Start and stop the schedule
-Countdown timer until next break
-Countdown timer for duration of break
-Push notifications when the break starts and stops

#Technologies used
-Swift
-SwiftUI
-IOS local Notifications
-Docker

#How it works
1. Launch the app
2. Select how often you want to take breaks
3. Select how long you want the breaks to be
4. Start the schedule
5. The app counts down to the next break
6. When the timer reaches zero, you will receive a notification
7. The break timer begins counting down
8. When the break ends, another notification will be sent
